## ✨ Features

### 🎬 Screens Included
- **Splash Screen** — Ripple animation, logo reveal, auto-loader bar (2.3s transition)
- **Login Screen** — Indian phone number validation (+91), shake error animation
- **OTP Screen** — 4-box auto-focus input, 30s countdown resend timer, spinner verify
- **Home Screen** — User greeting, promo banner, pickup/drop with swap, map placeholder, quick chips
- **Ride Options Screen** — Bike 🏍️ | Auto 🛺 | RapidoCab 🚗 with custom SVG icons, selectable cards
- **Booking Confirmation** — Pulsing ring animation, 4-step live progress tracker, driver card, ETA bar, SOS button
- **My Rides Screen** — Ride history with Completed / Cancelled status badges
- **Profile Screen** — Avatar, rating badge, wallet balance, settings menu

### ⚙️ Functional Highlights
- 📲 Single Page App (SPA) style navigation — no page reloads
- ✅ Phone number validation (Indian format: starts 6–9, exactly 10 digits)
- 🔢 OTP box auto-advance & backspace handling
- 🔄 Pickup ↔ Drop swap button
- 🎯 Ride card selection updates fare & confirm button label live
- ⏳ Booking spinner → animated step-by-step progress
- 🧭 Bottom navigation bar synced across all screens

### 🎨 UI / UX Highlights
- ⚡ Yellow `#FFD700` × Black `#111111` Rapido brand theme
- 🎯 Custom hand-drawn SVG vehicle illustrations (Motorcycle, Auto Rickshaw, Sedan)
- 💫 Smooth CSS screen slide transitions
- 🌊 Ripple, pulse, float, ring animations
- 🔤 Google Fonts — **Poppins** throughout
- 📐 Mobile-first layout (max-width 400px phone frame)
- 🌑 Soft shadows, rounded corners, minimal card design
- 🖱️ Micro-interactions: button hover, active scale, shake errors

## 🏃 Getting Started

### Run Locally

1. **Clone the repo**
   ```bash
   git clone https://github.com/Abdulkalam143/rapido-clone.git
   cd rapido-clone
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html directly
   open index.html

   # Or use Live Server in VS Code (recommended)
   # Right-click index.html → Open with Live Server
   ```

> ✅ No build tools, no npm install, no dependencies. Just open and run!
